# vue-elementui

## Create .env
Untuk mendapatkan API KEY silahkan access [disini](https://www.pexels.com/id-id/api/)
```
VUE_APP_NAME=BOILER VUE NEW
VUE_APP_BASE_URL=http://localhost:8080
VUE_APP_BASE_PROXY=https://api.pexels.com/v1/
VUE_APP_API_KEY=#############
```

## Project setup
```
npm install
```
